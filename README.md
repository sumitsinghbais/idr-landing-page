# Institute of Digital Risk – Landing Page

A responsive landing page designed for the **Institute of Digital Risk (IDR)**.
This project demonstrates modern frontend development practices using **HTML, CSS, and JavaScript**.

## 📌 Project Overview

The **Institute of Digital Risk (IDR)** is an industry-led training and deployment institute focused on digital, cyber, and technology risk roles.

This website presents the institute's mission, services, and community through a modern, responsive landing page.

---

## ✨ Features

* Responsive design (mobile & desktop)
* Sticky navigation bar
* Smooth scrolling navigation
* Animated hero section
* Interactive service cards
* Gradient hover effects
* Card tilt animation
* Pipeline model visualization
* Contact form UI
* Modern UI/UX design

---

## 🧰 Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **Flexbox & Responsive Layout**
